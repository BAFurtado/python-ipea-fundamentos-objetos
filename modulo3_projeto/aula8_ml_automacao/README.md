### Panorama rápido e básico de ML in Python

### Bibliotecas básicas

- **scikit-learn**: Central. Otimizada, fundamentada. 
Regressão linear (do ponto de vista computacional), Árvores de decisão, (random forest). Clustering, SVM

- pandas & NumPy: manipulação dados. 

- Matplotlib & Seaborn: visualização. Seaborn: esteticamente mais agradável

### Para Deep Learning 

- TensorFlow / Keras: padrão da indústria. Exige ambiente próprio de instalação. Keras roda em cima/utilizando recursos do TensorFlow (Google)

- PyTorch: flexível, pythonica. 

Veja ainda:

- XGBoost / LightGBM: eficientes: gradiente boosting

- Hugging Face Transformers: repostitório de modelos the Natural Language Processing (NLP). Vários exemplos, modelos pré-treinados, tutoriais.
